📡 What This App Does

Scan My Network is a lightweight utility that scans your local area network (LAN) for active devices using ARP discovery.

🧠 How It Works

  1. Sends ARP requests to your subnet (e.g., 192.168.1.0/24)
  2. Receives replies from active devices
  3. Captures IP, MAC, hostname, and marks them as online
  4. Stores results in a local SQLite database
  5. Displays data in a user-friendly web dashboard

🛠️ Technologies Used

  • 🐍 Python 3 — backend scripting
  • 🌐 Flask — lightweight web framework
  • 🗄 SQLite — embedded local database
  • 🎨 HTML & CSS — responsive web interface

🔐 Security & Privacy

This tool performs non-intrusive scans. All data stays local on your machine. Nothing is sent externally.

⚠️ Limitations

  • Only works within your current LAN (no external network scanning)
  • Some devices may not respond to ARP (e.g., due to firewall or stealth mode)
